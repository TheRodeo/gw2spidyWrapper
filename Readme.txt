This is a simple Wrapper for the Guild Wars 2 gw2spidy API written in Java.

Calling a method returns a String with the results or throws a SpidyException if no results were found.

For further details visit: https://github.com/rubensayshi/gw2spidy/wiki/API-v0.9#wiki-item-list
or: http://www.gw2spidy.com/
